# JLTC
JLTC is a control program for self-propelled robots.
Reference Book ISBN: 978-4-297-11299-8

# Environment
- Model: Raspberry Pi 4 Model B Rev 1.2
- OS: Raspbian GNU/Linux 10 (buster)

# Dependencies
- WiringPi
- OpenCV
- Picamera
